Data Loader

4 Access structured data function: getData()

getData function returns as output a list of items. The following are the principal ones:

  • arrayAssociativo: character array containing all the event labels
obj.out$arrayAssociativo
##  [1] "BEGIN"             "END"               "Medical Visit"     "Biopsy"           
##  [5] "total resection"   "MedicalVisit"      "death"             "chemotherapy"     
##  [9] "Imaging"           "partial resection" "radiotherapy"

“BEGIN” and “END”
